# m.protuberances.ru #


Серверный рендеринг и Redux

### ES6 style ###

* react
* react-router
* express
* Redux
* Webpack
* babel
* eslint
* esdoc

## Setup
```
npm install
```
## Development
```
npm run dev
```
## Production
* Build app
```
npm run build
```
* Run server
```
npm run start
```
## test 

```
npm test
```
## lint
* using ESlint

```
npm run lint
```

```
#!npm run doc
```
