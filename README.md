# m.protuberances.ru #


Серверный рендеринг и Redux

### ES6 style ###

* react
* react-router
* express
* Redux
* Webpack
* babel
* eslint
* esdoc

## Setup
```
npm install
```
## watch and autobuild
```
npm run watch
```
## run server
```
npm start
```
## manual build
* create bundle.js
```
npm run build
```
## test 

```
npm test
```
## lint
* using ESlint

```
npm run lint
```
* using ESDoc
```
#!npm run doc
```

* Вы можете посмотреть документацию на http://localehost:3010/esdoc/
